A new work in which the state of the art results on image colorization are obtained. The authors used the previous SOTA model, which used to predict the colorization separately for the whole image and individual objects, and combined the predictions using the attention-based fusion module.

Instance-aware Image Colorization

https://arxiv.org/abs/2005.10825

https://ericsujw.github.io/InstColorization

https://github.com/ericsujw/InstColorization

Authors have constructed the pipeline on idea that the model is difficult to train so that it worked well both on separate instances, and on the big general plans and a background.

Therefore they offered to do these two transformations separately, and then to fuse them by means of fusion module.

Pipeline is trained in three stages:

  1. First, the full-image network is trained.
  1. Then the instance network is trained (it receives an object cut out with Mask R-CNN).
  1. Then the fusion module is trained.

Full-image network and instance network architecturally repeat the model from [Real-Time User-Guided Image Colorization with Learned Deep Priors] (https://arxiv.org/abs/1705.02999).

Fusion module is a set of modules for several layers of full-image and instance networks, combining them as follows:

  • the full image feature tensor is passed through three convolution layers to obtain a weight map for the full image
  • for each found object the feature tensor is also passed through three convolution layers, the resulting map of weights is resized to the original image size (values outside bbox are filled with zeros)
  • all received tensors are stacked and softmax is applied to them.
  • all feature tensors are summed up with the weights obtained in the previous stage. These features are used to obtain a final prediction.

The model showed itself on metrics and visually better than all competitors, both at full image and at the instance level. Speed reaches 5-7 FPS on 2080Ti at 256x256 image resolution.

The model fails in case of detector errors or large number of objects.
