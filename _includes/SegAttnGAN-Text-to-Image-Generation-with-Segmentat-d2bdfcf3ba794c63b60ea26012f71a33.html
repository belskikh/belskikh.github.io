Improved architecture for text to image AttnGAN, the essence of which is that the input of the segmentation mask of the generated object is added to the image generator. Also developed variant with self-attention, when the mask is generated directly from the text using a separate generator.

SegAttnGAN: Text to Image Generation with Segmentation Attention

https://arxiv.org/abs/2005.12444

This architecture basically repeats AttnGAN architecture, where LSTM text encoder was used to get features from the text, and then the generator created the image from the resulting vector with the addition of noise vector.

The novelty is that a new module has been added that uses the mask of the generated object in the generator.

This mask is obtained in two ways: 1) simply take from the corresponding dataset, 2) use the additional generator for text to mask, which generates a mask from text.

As can be seen from the comparative results, this approach allowed to noticeably improve the quality of generation.
