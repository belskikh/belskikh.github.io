The new SOTA in the talking head generation, where special attention was paid to the separate reception of landmarks of the face, and also trained a separate network for the generation of blinking eyes.

Identity-Preserving Realistic Talking Face Generation

https://arxiv.org/abs/2005.12318

The approximate algorithm is this:

  • obtain person-independent landmarks using a separately trained model from DeepSpeech features (or, more precisely, predict the coordinate displacements of points in a standard landmark model of a face)
  • a blink is added to these landmarks by a separately trained network (the authors especially point out this innovation as the key to creating realism in the generated video).
  • then the landmarks translate back to the target face shape.
  • then the authors create a black and white image from the Landmark, connect it with the target face and feed it to an encoder-decoder generator that trains with LSGAN loss. The generator predicts the attention map and color map of the face, the color map is summed up with the target image using weights from the attention map.

Authors obtains SOTA on metrics and visually, they trained the models on a single NVIDIA Quadro GV100. Audio-to-landmark model - 6 hours, blink model - 3 hours, landmark-to-image - 2 days.
