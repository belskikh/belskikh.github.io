Work from Google, which presents the neural network architecture for tabular data, surpassing the previously dominant approaches based on tree ensembles, while maintaining the properties of interpretability and lightness. In addition, it has self-supervising pre-training, which opens up great opportunities for learning and learning on small data sets.

TabNet: Attentive Interpretable Tabular Learning

https://arxiv.org/abs/1908.07442

https://github.com/google-research/google-research/tree/master/tabnet

Unlike tree-based approaches, the TabNet input receives raw tabular data, and it learns through a normal gradient descent, learning good data representations.

The architecture uses a sequential attention mechanism that determines which features will be used during the current decision step, which allows to get an interpretable prediction.

To select features for the next step, the architecture uses a learnable sparse mask, which is elementwise multiplied by features embeddings in order to obtain weighted embeddings.

Discrete features are classically given as a table of learning embeddings. The features themselves are run through a special feature transformer module (has nothing to do with the Transformer architecture).

In self-supervised pretraining mode, the model learns to predict missing features from the input table, where these features are randomly masked.

The model shows up on par or better than the current SOTA models for tabular data.
